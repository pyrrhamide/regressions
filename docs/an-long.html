5.2 Modèles linéaires

Modèle de base: \[y_{i,t} = \alpha_i + \beta_i k_{i,t} + \gamma_i n_{i,t} + \epsilon_{i,t}\]

lineaire <- lm(var_dep ~ var_indep, data=d)

Modèle pooled: \[y_{i,t} = \alpha + \beta k_{i,t} + \gamma n_{i,t} + \epsilon_{i,t}\]

pooled <- plm(var_dep ~ var_indep, 
              data=d, 
              index=c("var_individu","var_temporelle"),
              model="pooling")

Modèle à effet fixe: \[y_{i,t} = \alpha_i + \beta k_{i,t} + \gamma n_{i,t} + \epsilon_{i,t}\]

effet_fixe <- plm(var_dep ~ var_indep,
                  data=d,
                  index=c("var_individu","var_temporelle"),
                  model="within")
